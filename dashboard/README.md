# Composant Dashboard — Visualisation Graphique Dynamique

> 🔧 À implémenter

Ce dossier contiendra le code de l'interface graphique qui :
- Affiche le graphe de connexions (Utilisateurs, Vendeurs, Produits)
- Se rafraîchit automatiquement (fréquence paramétrable, ex: toutes les 5s)
- Distingue visuellement les nœuds par type (couleur/symbole)
- Représente les arêtes typées (AIME, VOUT, ACHAT) et leurs poids
